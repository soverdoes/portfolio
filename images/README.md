# 이미지 교체 가이드 — 최재성 포트폴리오

아래 파일명 그대로 이미지를 넣으면 코드 수정 없이 바로 반영됩니다.
모든 파일은 **JPG**, 무게는 장당 300KB 이하 권장(웹 로딩 속도). 세로형 폰은 PNG(투명) 가능.

## 2. 프로젝트 상세 — `images/projects/<슬러그>/` (프로젝트당 4장)
Projects 케이스 스터디. `-cover`는 홈 시그니처 스포트라이트에도 재사용(CPASS·바로산책·GK OPS).

프로젝트당 파일:
- `<슬러그>-cover.jpg` — 대표 커버 (16:9, 1600×900)
- `<슬러그>-01.jpg` — 상세 컷 1
- `<슬러그>-02.jpg` — 상세 컷 2 (폰 목업이면 세로형)
- `<슬러그>-03.jpg` — 풀블리드 컷 (와이드, 1600×720)

슬러그 8종: `cpass · baro · gkops · sandan · icya · poca · fitnavi · groundk`
