01 / CPASS

CPASS

보유 중인 가상자산을 한 곳에 모아 쉽고 간편하게 운용하는 지갑 앱. 다크 톤과 글래스모피즘으로 전문성·미래지향적 무드를 완성했습니다.

Role
참여 2명 · 기여도 80%
Client
(주)콰가
Year
2022
Tools
Figma
Type
App · Web

Problem

코인 스왑과 자산 관리는 노출되는 정보가 많아 복잡하고, 다양한 연령층에게 진입장벽이 높았습니다.

Solution

정보의 중요도에 따라 컬러·폰트 레벨을 부여하고, 코인 심볼 컬러를 그래프에 그대로 사용해 이름을 읽지 않아도 색으로 구별되게 했습니다. 하단 슬라이드 형태로 Depth를 줄여 이탈률을 낮췄습니다.

02 / BARO

바로산책

바쁜 반려인을 위한 반려동물 산책 긱플랫폼. 당일 신청이 가능한 자동화 매칭 시스템으로 걱정을 덜어줍니다.

Role
1명 · 기획·설계·디자인 100%
Client
(주)브이아이펫
Year
2022
Tools
Adobe XD
Type
App

Problem

잦은 야근·외박으로 반려견 산책이 어렵고, 비전문 펫워커에 대한 불신과 복잡한 가입·예약 과정이 진입을 막고 있었습니다.

Solution

사용자 인터뷰에서 출발해 반려견 정보·접종여부 등록, 위치기반 실시간 산책 확인, 연락처 없이 가능한 매니저 채팅을 설계했습니다. SNS 로그인과 단계형 예약으로 가입·결제의 Depth를 최소화했습니다.

03 / GK OPS

GK OPS

배차·관제 시스템 대시보드 리디자인. 기획팀과 개발팀의 상충하는 요구를 조율해 정보량과 속도를 동시에 잡았습니다.

Role
UX 리디자인 · 이해관계자 조율
Client
(주)그라운드케이
Year
2023
Tools
Figma
Type
B2B Dashboard

Problem

기획팀은 20개 이상의 정보를 한 화면에, 개발팀은 로딩 속도 때문에 핵심만 보여주길 원했습니다. 양쪽 모두 물러서지 않아 일정이 지연됐습니다.

Solution

정보를 ‘자주 보는 것 / 가끔 보는 것’으로 재분류하고 단계별 로딩을 제안했습니다. Figma로 3단계 로딩 구조를 실제 화면으로 시각화해 양팀을 설득했고, 이 구조는 현재 사내 모든 B2B 대시보드의 표준이 되었습니다.
